**Level**: Bachelor 2 (Licence 2) - Computer Science  
**Hours**: 36h (2024-2025)  
**Course Coordinator**: Karim Tabia  
**Language**: SQL

## Course Content
* Database fundamentals: tables, identifiers, primary and foreign keys, integrity constraints
* Data modeling: entity-relationship and relational models
* SQL language: queries, data manipulation, and database management
* Student guidance on SAÉ (Learning and Evaluation Situations) projects
* Project supervision and grading

## Skills Developed
Students learn to design normalized database schemas and write efficient SQL queries for real-world data management scenarios.
