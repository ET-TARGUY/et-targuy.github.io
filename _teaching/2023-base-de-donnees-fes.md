**Level**: Bachelor 1 (Licence 1) - Biology, Chemistry, Geology (BCG)  
**Hours**: 72h (2023-2024)  
**Course Coordinator**: Younes Dhassi  
**Language**: SQL

## Course Content
* Database fundamentals: tables, identifiers, primary and foreign keys, integrity constraints
* Data modeling: entity-relationship and relational models
* SQL language: queries, data manipulation, and database management
* Test preparation and grading

## Skills Developed
Introduction to database concepts for science students, enabling them to manage and query scientific data efficiently using relational database systems.
