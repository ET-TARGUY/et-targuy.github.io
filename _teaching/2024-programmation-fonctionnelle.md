**Level**: Bachelor 1 (Licence 1) - Mathematics and Computer Science  
**Hours**: 24h (2022-2023), 24h (2024-2025)  
**Course Coordinator**: Fabien Delorme  
**Language**: Haskell

## Course Content
* Fundamental concepts of functional programming
* Recursion and lazy evaluation
* Higher-order functions and lambda expressions
* Pure functional programming paradigm

## Skills Developed
Students develop a strong foundation in functional programming concepts, learning to think recursively and work with immutable data structures using Haskell.
